Step3 Dynamic SVG - 3 Concepts Rendering Test

1. GDN (Gated Delta Network)

Core: Learnable gate dynamically adjusts fusion ratio of history state and current input, achieving precise memory at linear complexity.

2. Ring Attention

Core: Distribute sequence across GPUs in a ring, pass KV blocks in circular pattern so each GPU computes local attention while collectively covering the full context.

3. LoRA (Low-Rank Adaptation)

Core: Freeze original weights, add trainable low-rank bypass (A*B) alongside, achieving efficient fine-tuning with minimal parameters.
